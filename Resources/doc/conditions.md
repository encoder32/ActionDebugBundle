Dump Condition
--------------

**Aliases:** dump_true, dump_false

**Description:** Dump current context into Debug Toolbar and return TRUE or FALSE result.

**Options:**
 - Label

**Code Example**

```yml
- @dump_true: 'my label, true'
- @dump_false: 'my label, false'
```
